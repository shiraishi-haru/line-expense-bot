# LINE公式アカウントのメッセージ設定

LINE公式アカウントを作成したあと、**挨拶メッセージ**と**応答（Webhook）**の設定を行います。このBotでは「応答」はすべてサーバー（Webhook）が行うため、LINE側の自動応答はオフにします。

---

## 1. 挨拶メッセージ（友だち追加時に表示）

友だち追加した直後に、ユーザーに表示するメッセージを設定します。

### 設定手順

1. [LINE Developers](https://developers.line.biz/) にログインし、該当の**プロバイダー** → **チャネル（Messaging API）**を開く
2. 左メニューまたはタブで **「Messaging API」** を選択
3. 下にスクロールし、**「Greeting messages」（あいさつメッセージ）** の **「設定」** をクリック
4. **「あいさつメッセージを有効にする」** をオンにする
5. **メッセージ**欄に、次のような文言を入力する（例）：

   ```
   はじめまして。交通費精算Botです。
   メッセージを送信すると、招待コードの入力から案内します。
   ```

6. **「更新」** で保存

### 文言の例（コピー用）

- **短い例**  
  `交通費精算Botです。メッセージを送ると招待コードの入力をお願いします。`

- **少し詳しい例**  
  `はじめまして。交通費精算Botです。初回は招待コードとお名前の入力をお願いします。登録後は交通費をそのまま送信できます。`

※ 挨拶メッセージは**友だち追加時のみ**表示されます。その後の「招待コードを入力してください」などは、すべて**Webhook（このBotのサーバー）**が返します。

---

## 2. 応答メッセージ（Webhook で返すため自動応答はオフ）

このBotでは、ユーザーが送ったメッセージへの返信は**すべてWebhook（あなたのサーバー）**が行います。  
そのため、LINEの「自動応答メッセージ」は**オフ**にします。

### 設定手順

1. 同じく **LINE Developers** → 該当チャネルの **「Messaging API」** を開く
2. **「Response settings」（応答設定）** を探す
3. **「Auto-response messages」（自動応答メッセージ）** を **オフ（Disabled）** にする  
   - オンになっていると、LINEが定型文で返信し、Webhookが返したメッセージと二重になったり、Webhookの返信が届かなくなることがあります
4. **「Use webhook」** を **オン（Enabled）** にする（すでにWebhook URLを設定している場合はオンになっています）
5. **Webhook URL** に、あなたのサーバーのURLを入力する  
   - ローカル確認時: `https://xxxx.ngrok-free.app/webhook` など  
   - 本番（Render）: `https://あなたのサービス名.onrender.com/webhook`
6. **「Verify」** を押して成功すれば、ユーザーが送ったメッセージがすべてあなたのサーバーに送られ、サーバーが返したメッセージがユーザーに届きます

### まとめ（応答まわり）

| 項目 | 設定 |
|------|------|
| あいさつメッセージ | オン。友だち追加時に表示する文言を設定 |
| 自動応答メッセージ | **オフ**（Webhook が返すため） |
| Use webhook | **オン** |
| Webhook URL | あなたのサーバーURL + `/webhook` |

---

## 3. チャネル基本設定（任意）

**LINE Developers** の **「Basic settings」** では、次のような表示用の設定ができます。

- **Channel name** … ユーザーに表示される公式アカウント名
- **Channel description** … アカウントの説明文
- **Channel icon** … アイコン画像
- **Welcome message when users add your LINE Official Account as a friend** … ここで設定する内容が、管理画面の「あいさつメッセージ」と連動している場合があります（製品により表記が異なります）

実際の「友だち追加時メッセージ」は、**Messaging API** の **Greeting messages** で設定した内容が使われます。

---

## 4. このBotでのメッセージの流れ（参考）

1. **友だち追加** → LINEが **挨拶メッセージ**（上記で設定した文）を表示
2. **ユーザーが何か送信** → LINEがその内容を **Webhook** に送信
3. **サーバー（このBot）** が、登録状態に応じて返信  
   - 未登録: 「招待コードを入力してください」→「フルネームを漢字で…」→「登録が完了しました」
   - 登録済: 交通費の登録・履歴・月別集計・完了報告など

挨拶メッセージ以外の**すべての返信**は、このBotのコード（`app/line_handlers/messages.py` と `webhook_handler.py`）で決まっています。文言を変えたい場合は、そちらのファイルを編集してください。
